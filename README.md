# Nome do Projeto

API para gestão de títulos a pagar

## Tecnologias
- Node.js
- PostgreSQL
- Express

## Configuração
1. Instale as dependências: `npm install`
2. Crie um arquivo `.env` baseado no `.env.example`
3. Execute: `npm start`

## Endpoints
- `GET /titulos` - Lista títulos abertos
- `POST /faturar` - Fatura títulos
